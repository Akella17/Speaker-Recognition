# Speaker-Recognition

Uses a 3 layer stacked LSTM architecture with 100 hidden units in each LSTM cell. The network is trained over Voice Conversion (VCC) 2016 dataset.

### Model Architecture

3 Layer Stacked LSTM network : 100 hidden units each
